# Installation
**Install python**
```bash
apt update && apt install git python3 python3-pip -y
```
**Clone repo and install requirements**
```bash
git clone https://github.com/itzlayz/Linto && cd Linto && pip install -r requirements.txt
```
**Run linto**
```bash
python3 -m Linto
```
